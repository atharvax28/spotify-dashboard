# Spotify Analytics Dashboard

A dashboard for visualizing Spotify listening data — charts and breakdowns of tracks, artists, and trends, built with React and Recharts.

🔗 **Live:** https://spotify-dashboard-wine.vercel.app

---

## Highlights

- **React + TypeScript + Vite**
- **Recharts** for interactive data visualizations
- Clean, component-driven dashboard layout with `lucide-react` icons

## Tech stack

`React` · `TypeScript` · `Vite` · `Recharts`

## Getting started

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build
```
